# 深度学习实验合集

本仓库包含深度学习基础实验任务代码与文档。

## 任务介绍

### Task2：MLP 数据拟合与可视化
读取 CSV 文件，使用 MLP 模型进行数据拟合，并可视化不同训练轮次的拟合效果。

### Task3：CNN 图像分类（CIFAR-10）
基于 CNN 实现 CIFAR-10 图像分类，掌握卷积网络基本流程。

### Task4：CIFAR-10 模型优化（ResNet+Warmup+Mixup）
- 残差结构改进模型
- Warmup 学习率 + Mixup 数据增强
- 对比实验与性能分析
- 详细说明：[task4.md](task4.md)

### Task5：Mask R-CNN 行人实例分割
- 使用 Mask R-CNN 实现检测 + 分割
- 数据集：PennFudanPed
- 结果可视化展示
- 详细说明：[task5.md](task5.md)
